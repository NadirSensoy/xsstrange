Cross-Site Scripting (XSS)

High Risk

Cross-Site Scripting (XSS) attacks are a type of injection where malicious scripts are injected into trusted websites. These attacks occur when an attacker uses a web application to send malicious code to different end users.

Impact

  • Cookie theft and session hijacking
  • Keylogging and data theft
  • Phishing attacks
  • Website defacement

Common Vectors

  • User input fields
  • URL parameters
  • HTTP headers
  • File uploads
